How to Install and Use Discord

Install | Use | Video

Installation Instructions

  1. Navigate to the Discord Website.
  2. Click on the Download button at the top of the page.
  3. Select "Dowload for Windows" button. Navigate down the page if you are on a Mac or other device.
  4. Finish installation and open Discord.

Usage Instructions

  • Discord can be open on either a computer or your phone.
  • Utilize Discord to chat with classmates, friends, or with your professor.
